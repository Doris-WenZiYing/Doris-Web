# Doris Wen — Portfolio

Vite + React 19，中英文切換，Gazi 風格 TopNav。

## 快速開始

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # 打包到 dist/
```

## 放照片

把你的照片存成 `public/profile.png`

## 作品集封面圖（可選）

把截圖放進 `public/projects/`，然後在 Projects.jsx 的 bg 欄填路徑：
`bg: "/projects/iosclub.png"`

## 部署

把 `dist/` 資料夾上傳到 Vercel / Netlify 即可
