# 我的作品集

Vite + React 19 作品集，中英文切換，灰調風格。

## 快速開始

```bash
npm install
npm run dev
```

## 部署

```bash
npm run build
# 把 dist/ 資料夾上傳到 Vercel / Netlify / GitHub Pages
```

## 客製化

| 要改的地方 | 檔案 |
|---|---|
| 姓名、email、社群連結 | `src/components/SidebarNav.jsx` |
| About 文字 | `src/components/About.jsx` |
| 工作經歷 | `src/components/Experience.jsx` |
| 作品 & 服務 | `src/components/Projects.jsx` |
| 聯絡資訊 | `src/components/Contact.jsx` |
| 顏色 / 字型 | `src/styles/Global.css` |
